# 🔐 SOC Analyst Level 1 – Detection Engineering Portfolio

This repository contains hands-on SOC Level 1 detection projects implemented using Elastic SIEM.

---

# 📌 Projects Overview

## 🛡 Project 1 – Windows Failed Login Monitoring
- Monitored Event ID 4625
- Built dashboard in Kibana
- Analyzed failed authentication attempts

---

## 🚨 Project 2 – Brute Force Detection (Threshold Rule)
- Created threshold-based alert
- Detected multiple failed logins from same source IP
- Time-window correlation logic

---

## ⚡ Project 3 – Suspicious PowerShell Execution Detection
- Monitored process events
- Detected usage of -EncodedCommand
- High severity alert configuration

---

## 👑 Project 4 – Local Admin Account Creation Detection
- Monitored Event IDs 4720, 4732, 4672
- Detected privilege escalation attempts
- High risk rule creation

---

## 🌐 Project 5 – Port Scan Detection
- Detected multiple connection attempts
- Identified reconnaissance behavior
- Created threshold detection rule

---

# 🔥 Advanced Project – End-to-End Attack Chain Detection

Simulated and detected a complete multi-stage attack:

1. Port Scan (T1046)
2. Brute Force (T1110)
3. Valid Account Login (T1078)
